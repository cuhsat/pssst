Pssst Server
============
If you are using Heroku, the server can easily be deployed using this
custom [buildpack](https://github.com/cuhsat/heroku-buildpack-pssst).

Usage
-----
```
$ npm start [option]
```

The default server port is `62421` and can be changed in `config.json`.
