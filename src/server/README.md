Server
======
Pssst server.

Usage
-----
```
$ npm start [option]
```

The default server port is `62421` and can be changed in `config.json`.

Deploy
------
If you are using Heroku, this server can easily be deployed using this custom
[buildpack](https://github.com/cuhsat/heroku-buildpack-pssst).
